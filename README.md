# React Reusable Components Library Boilerplate

Useful reusable components library built with create-react-app with Storybook, SCSS and Jest.

| Built with | 
|---|
| Storybook  | 
| SASS |   
| Jest |


## Installation

Fork and clone this repository

```bash
git clone https://github.com/saraahso/react-component-library.git
```

Install depedencies

```bash
npm i
```

## Usage

Visualize library with Storybook

```bash
npm run storybook
```
